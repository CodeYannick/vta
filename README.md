# VTA Tour MVP Prototype

This is the interactive prototype for the VTA Tour MVP Mini Program.

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Lucide React (Icons)
- React Router DOM

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at the URL shown (usually http://localhost:5173).

## Features Included
- **Home Page**: Focused on "Race to Finals" banner, quick tournament access, and grayed-out Training module.
- **Tournaments**: List view with filters (Registration/Ongoing/Completed).
- **Career**: "Constellation" visual (static), tier progress bar, and core stats.
- **My Page**: Player identity card (Rank/Tier), essential service shortcuts.
